# Data Network Protocol Simulation (DNS + FTP)

## Overview

This project implements and analyzes key application-layer network protocols using the OMNeT++ simulation framework. It focuses on the design, behavior, and interaction of:

- **DNS (Domain Name System)**
- **FTP (File Transfer Protocol: RETR & STOR)**
- **Integrated DNS → FTP real-world workflow**

The simulation provides packet-level insight using ARP, TCP, UDP, and caching mechanisms, validated through Wireshark captures and OMNeT++ logs.
